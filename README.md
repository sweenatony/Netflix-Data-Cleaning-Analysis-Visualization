# Netflix-Data-Cleaning-Analysis-Visualization

## 📌 Project Overview  
This project analyzes Netflix movies and TV shows using Python and Tableau to uncover trends in content distribution, ratings, genres, and release patterns.

The goal is to understand how Netflix content has evolved over time and derive insights to support data-driven content strategy decisions.


## 🎯 Objectives  
- Analyze distribution of Movies vs TV Shows  
- Identify top genres on Netflix  
- Understand content trends over time  
- Analyze rating distribution  
- Explore country-wise content production  


## 🛠️ Tech Stack  
- **Python** – Data Cleaning & Exploratory Data Analysis  
- **Pandas, NumPy** – Data Manipulation  
- **Matplotlib, Seaborn** – Data Visualization  
- **Tableau** – Dashboard Development  
- **Excel / CSV** – Data Source  


## 📂 Dataset  
- Netflix Movies & TV Shows Dataset  


## 🧠 Approach  

### 1. Data Cleaning & Preprocessing
- Handled missing values and null entries  
- Standardized genres, country names, and date formats  
- Prepared clean dataset using Python  

### 2. Exploratory Data Analysis (EDA)
- Analyzed content distribution by type, genre, and country  
- Identified trends over time  
- Explored rating patterns and popularity  

### 3. Visualization & Dashboard
- Created visualizations using Python libraries  
- Built an interactive dashboard in Tableau  


## 📊 Key Insights  
- Movies dominate Netflix content (~70%), while TV Shows make up ~30%  
- Most content is rated **TV-MA** and **TV-14**  
- Top genres include **Drama, Documentaries, and Stand-up Comedy**  
- Content production increased rapidly after 2015 and peaked around 2019–2020  
- USA produces the highest content, followed by India and the UK  


## 🚀 Business Recommendations  
Based on the analysis, the following recommendations can help improve content strategy and user engagement:

- Focus on high-performing genres such as Drama, Documentaries, and Stand-up Comedy  
- Expand content production in high-demand countries like India and the UK  
- Invest more in TV Shows as their popularity is growing  
- Prioritize high-rated content (TV-MA, TV-14) to align with audience preferences  
- Maintain a consistent content release strategy to sustain growth
